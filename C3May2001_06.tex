\JGWitem{May 2001, \#6}
 For a multiple decrement model on (60):
\begin{enumerate}
\item $\mu_{60}^{(1)}(t)$ follows the Illustrative Life Table.
\item $\mu_{60}^{(\tau)}(t) = 2\mu_{60}^{(1)}(t)$ for $t\geq 0$.
\end{enumerate}
Calculate $\prepostsubsup{11|}{q}{60}{(\tau)}$, 
the probability that decrement occurs during the 11th year.
\showMC{\begin{description}
\item[(A)] 0.03
\item[(B)] 0.04
\item[(C)] 0.05
\item[(D)] 0.06
\item[(E)] 0.07
\end{description}}
\showsol{\bsoln
\showMC{\underline{ANS: (B)}}
If $\mu_x(t)$ is a given force-of-mortality function, we know that 
\[
   \prepostsub{t}{p}{x} = e^{-\int_0^t \mu_x(s)\,ds}
\]
and so if we double $\mu$ then we square the survival function:
\[
   e^{-\int_0^t 2\mu_x(s)\,ds} = e^{-2\int_0^t \mu_x(s)\,ds} = \left(e^{-\int_0^t \mu_x(s)\,ds}\right)^2 =  \left(\prepostsub{t}{p}{x}\right)^2.
\]
So
\bears
   \prepostsubsup{11|}{q}{60}{(\tau)} &=& \prepostsubsup{10}{q}{60}{(\tau)}q_{70}^{(\tau)} = \prepostsubsup{10}{q}{60}{(\tau)}(1-p_{70}^{(\tau)}) \\
                                      &=& \left(\frac{S(70)}{S(60)}\right)^2\left(1-\left[\frac{S(71)}{S(72)}\right]^2\right) = 0.042612
\eears
\esoln}
% (ILT(70)/ILT(60))*(1-(ILT(71)/ILT(70))^2)
