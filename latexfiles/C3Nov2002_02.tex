\JGWitem{Nov 2002, \#2}
%(same as M0905 \#34)
 For a triple decrement table, you are given:
\begin{enumerate}
\item $\mu_x^{(1)}(t) = 0.3$ for $t>0$, 
\item $\mu_x^{(2)}(t) = 0.5$ for $t>0$, 
\item $\mu_x^{(3)}(t) = 0.7$ for $t>0$, 
\end{enumerate}
Calculate $q_x^{(2)}$.
\showMC{\begin{description}
\item[(A)] 0.26
\item[(B)] 0.30
\item[(C)] 0.33
\item[(D)] 0.36
\item[(E)] 0.39
\end{description}}
\showsol{\bsoln
\showMC{{\bf Ans: (A)}}

We have $\mu^{(\tau)}_x(t) = 1.5$, and:
\bears
 q_x^{(2)} &=& \int_0^1 \prepostsub{s}{p^{(\tau)}}{\!\!\!\!\!x}
  \mu_x^{(2)}(s)\,ds 
  = 0.5\int_0^1 e^{-1.5 s}\,ds = {0.5\over 1.5}(1-e^{-1.5})=0.25896
\eears

\esoln}
