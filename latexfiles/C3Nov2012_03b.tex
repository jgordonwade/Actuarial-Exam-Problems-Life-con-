\JGWitem{C3 Nov 2012 \#03-b}   % \S2 Single Life Survival
You are given:
\begin{enumerate}
\item $S_0(t) = \left(1-\frac{t}{\omega}\right)^{1/3}$ for $0\leq t \leq\omega$.
\item $\mu_{35} = \frac{1}{210}$
\end{enumerate}
Find  $\omega$.

\showsol{\bsoln
We need to find $\omega$. Using the relation $\mu_{x} = -S'(x)/S(x)$, \\
we find that $\mu_{x} = \dfrac{1}{3(\omega-x)}$, so that 
$\mu_{65}=1/210$ implies $3(\omega-35) = 210$ or \underline{$\omega=105$.}



\esoln}
