\JGWitem{May 2005 \#14}
 For a special fully discrete 2-year endowment insurance of 1000 on (x), 
you are given:
\begin{enumerate}
\item The first year benefit premium is 668.
\item The second year benefit premium is 258.
\item $d = 0.06$
\end{enumerate}
Calculate the level annual premium using the equivalence principle.
\showMC{\begin{description}
\item[(A)] 469
\item[(B)] 479
\item[(C)] 489
\item[(D)] 499
\item[(E)] 509
\end{description}}
\showsol{\bsoln
Since ``benefit premium'' means premiums computed so that $\Expect{L}=0$, we have:
\bears
   \Expect{L} = 0 &=& 1000A_{x:\halfbox{2}} - 668 - 258vp_x \quad\quad\mbox{($v=1-d$)}.
\eears
We need to find $p_x$, using this equation and information about $A_{x:\halfbox{2}}$:
\bears
  A_{x:\halfbox{2}} &=& \termify{A_{x:\halfbox{2}}} + \prepostsub{2}{E}{x} \\
   &=& \Bigl[vq_x + v^2\prepostsub{1|}{q}{x}\Bigr] + v^2\prepostsub{2}{p}{x} \\
   &=& v(1-p_x) + v^2\Bigl[\prepostsub{1|}{q}{x} + \prepostsub{2}{p}{x}\Bigr] \\
   &=& v(1-p_x) + v^2p_x  \\
   &=& v + p_x(v^2-v) = v - vdp_x
\eears
So we have 
\bears
   0 &=& 1000\left[v - vdp_x\right] - 668 - 258vp_x \\
   668 - 1000v &=& p_x\left(1000vd - 258v\right) \\
   -272.00 &=& -298.920 p_x \quad\quad\Longrightarrow \underline{p_x = 0.90994}
\eears
So,
\bears
   A_{x:\halfbox{2}} &=&  v - vdp_x = 0.88868,  \\
 \mbox{and }\quad
    P_{x:\halfbox{2}} &=& \frac{dA_{x:\halfbox{2}}}{1-A_{x:\halfbox{2}}} = 0.47898
\eears
so the level annual premium is 478.98.
   

\esoln}
